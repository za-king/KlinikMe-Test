How run the development server:

clone service from [https://github.com/za-king/KlinikMe-Test-Service]

in service run :

```bash
npm install
#and
node index.js
```

in client run :

```bash
npm install
# and
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
